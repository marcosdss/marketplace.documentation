# Spike - LIONS 292

### Assinatura | Cancelamento de assinatura

**1. Introdução**

Que o cliente possa solicitar o cancelamento da Assinatura.

**2. Solução**

Disponibilizar um botão no My Account para o usuário cancelar a Assinatura via API Subscriptions.

**3. Diretrizes para o Desenvolvimento**

O My Account não possui código aberto, portanto, o botão será inserido via [Helmet](https://github.com/vtex-apps/render-runtime?tab=readme-ov-file#helmet) pelo repositório [oba.my-account](https://github.com/ObaHortifrutiDeveloper/oba.my-account).

Além disso, deve-se criar um app em NodeJS para consumir a API Subscriptions. Neste app iremos consultar a lista de clientes com assinaturas ativas, identificar o cliente atual e atualizar o campo **status** para **CANCELED**.

[List subscriptions filtering by some arguments.](https://developers.vtex.com/docs/api-reference/subscriptions-api-v3#get-/api/rns/pub/subscriptions)

[Update a specific subscription.](https://developers.vtex.com/docs/api-reference/subscriptions-api-v3#patch-/api/rns/pub/subscriptions/-id-)

_Este tema ainda está sendo tratado com a VTEX, podendo mudar a solução conforme for._
